# Requirement Risk Analyzer

A FastAPI-based application that analyzes software requirement documents and identifies:

- Missing requirements
- Project risks
- Client clarification questions
- Architecture concerns

## Tech Stack

- Python
- FastAPI
- OpenAI API
- pdfplumber
- Jinja2

## Current Features

- PDF upload
- TXT upload
- Document text extraction
- Requirement extraction using OpenAI

## Planned Features

- Gap Analysis Engine
- Risk Analyzer
- Question Generator
- Architecture Analyzer
- PDF Report Generation
